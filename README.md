# FAQ Bot 

An AI chatbot that answers frequently asked questions for a business or product using NLP.

## Features
- Answer common questions instantly
- Customizable FAQ database
- Simple frontend for interaction

##  Tech Stack
- Python
- Flask
- HTML/CSS/JS (Frontend)
- nltk / spaCy (for NLP)

## How It Works
1. The user sends a question.
2. The bot processes the question and finds the closest match from the FAQ database.
3. The bot replies with the relevant answer.

## How to Run
```bash
pip install -r requirements.txt
python app.py
